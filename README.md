SciTemper — Scientific Thinking Quiz Platform

SciTemper — Measure and Grow Your Scientific Mind. In today’s world, people are surrounded by information but often lack scientific thinking, which leads to misinformation and poor decision-making. SciTemper solves this by providing an adaptive platform that evaluates scientific literacy, critical thinking, and reasoning ability, and then guides users to improve. Our system measures six core dimensions, including conceptual understanding, critical thinking, statistical reasoning, scientific method, societal awareness, and misinformation resistance. Users take an interactive quiz with dynamic questions, instant feedback, and real-life logic-based scenarios. Based on performance, they are placed on a five-level scientific temper scale ranging from Novice to Scientific. The platform also provides visual analytics like score rings and radar charts to clearly show strengths and weaknesses. It detects misconceptions and explains them using scientific reasoning, ensuring deeper learning. Technically, the project is built using HTML, CSS, and JavaScript with dynamic quiz logic and real-time updates. What makes it unique is that it focuses on how people think rather than what they memorize. It can be used in education, exams, and awareness programs, with future scope including AI-based adaptation and mobile support. In conclusion, SciTemper is a step towards building a scientifically aware and rational society, thank you.

Tech Used: HTML, CSS, JavaScript, Node.js, Express, bcrypt
